
step 1 : create a new folder for playwright project  and open in vs code 

step 2 : create a python virtual env for this project using below command 
	 
	 python -m venv my_env 

step 3 : to activate the env use below

	my_env\scripts\activate

step 4 : once env active then inside run below command 

	pip install playwright to install playwright 

(optional) to upgrade playwright use this command -->pip install playwright -u
